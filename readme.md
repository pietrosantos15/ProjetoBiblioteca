# 📚 Bibliotech

O **Bibliotech** é um projeto de interface para uma biblioteca moderna, focado em oferecer uma experiência digital fluida para a descoberta de livros, gestão de coleções e participação em eventos comunitários.

## 🚀 Funcionalidades

* **Exploração de Catálogo**: Barra de pesquisa integrada para localizar livros por título, autor ou ISBN.
* **Novidades (New Arrivals)**: Secção dinâmica para exibição dos lançamentos mais recentes.
* **Gestão de Eventos**: Calendário de atividades como clubes de leitura e palestras com autores.
* **Serviços Digitais**: Informações sobre aluguer de audiolivros, reserva de salas de estudo e apoio à investigação académica.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica de toda a interface.
* **CSS3**: Estilização personalizada, incluindo layouts com Flexbox e design responsivo.
* **Tipografia**: Uso de fontes seguras para web como Arial e Helvetica para garantir legibilidade.

## 📂 Estrutura de Ficheiros

* `index.html`: Estrutura principal do site.
* `style.css`: Estilização e definições de layout.
* `/assets`: Recursos visuais, incluindo o logótipo, imagens de capa e ícones do projeto.
